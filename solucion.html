🚀 3 Opciones Disponibles

⭐ Opción 1: Comando Django (RECOMENDADO)

Ubicación: usuarios/management/commands/crear_empleados.py

Uso:

python manage.py crear_empleados --cantidad 10

Ventajas:

  • Integrado nativamente en Django
  • Evita duplicados automáticamente
  • Personalizable
  • Profesional y escalable

🔧 Opción 2: Script Python

Ubicación: crear_empleados_script.py

Uso:

python crear_empleados_script.py 10

Ventajas:

  • No requiere manage.py
  • Útil para automatización
  • Funciona en CI/CD

📦 Opción 3: Django Fixtures

Ubicación: fixtures/empleados.json

Uso:

python manage.py loaddata fixtures/empleados.json

Ventajas:

  • Datos precargados
  • Control total
  • Reproducible
